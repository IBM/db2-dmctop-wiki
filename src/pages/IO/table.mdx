# Metric shown

## Gauge


<hr />

### Rows r+w/s

The percentage of rows read from the table.

= ROWS_READ / ROWS_TOTAL 

**Source:** 

ROWS_READ = [MON_GET_TABLE](https://www.ibm.com/docs/en/db2/11.5?topic=functions-mon-get-table-get-table-metrics)(NULL, NULL, #MEMBER#).SUM([ROWS_READ](https://www.ibm.com/docs/en/db2/11.5?topic=reference-r#r0001317))

ROWS_TOTAL = [MON_GET_TABLE](https://www.ibm.com/docs/en/db2/11.5?topic=functions-mon-get-table-get-table-metrics)(NULL, NULL, #MEMBER#).(SUM([ROWS_READ](https://www.ibm.com/docs/en/db2/11.5?topic=reference-r#r0001317)) + SUM([ROWS_INSERTED](https://www.ibm.com/docs/en/db2/11.5?topic=reference-r#r0001313)) + SUM([ROWS_UPDATED](https://www.ibm.com/docs/en/db2/11.5?topic=reference-r#r0001314)) + SUM([ROWS_DELETED](https://www.ibm.com/docs/en/db2/11.5?topic=reference-r#r0001312)))

### Temp space

The total page size of system temporary data used on disk where `TBSP_CONTENT_TYPE` of table space equals to 'SYSTEMP'.

= (DATA_OBJECT_L_PAGES + INDEX_OBJECT_L_PAGES) * TBSP_PAGE_SIZE 

**Source:** 

DATA_OBJECT_L_PAGES = [MON_GET_TABLE](https://www.ibm.com/docs/en/db2/11.5?topic=functions-mon-get-table-get-table-metrics)(NULL, NULL, #MEMBER#).SUM([DATA_OBJECT_L_PAGES](https://www.ibm.com/docs/en/db2/11.5?topic=reference-d#r0059958))

INDEX_OBJECT_L_PAGES = [MON_GET_TABLE](https://www.ibm.com/docs/en/db2/11.5?topic=functions-mon-get-table-get-table-metrics)(NULL, NULL, #MEMBER#).SUM([INDEX_OBJECT_L_PAGES](https://www.ibm.com/docs/en/db2/11.5?topic=reference-i#r0059961))

TBSP_PAGE_SIZE = [MON_GET_TABLESPACE](https://www.ibm.com/docs/en/db2/11.5?topic=functions-mon-get-tablespace-get-table-space-metrics).[TBSP_PAGE_SIZE](https://www.ibm.com/docs/en/db2/11.5?topic=reference-t#r0007534)

TBSP_CONTENT_TYPE = [MON_GET_TABLESPACE](https://www.ibm.com/docs/en/db2/11.5?topic=functions-mon-get-tablespace-get-table-space-metrics).[TBSP_CONTENT_TYPE](https://www.ibm.com/docs/en/db2/11.5?topic=functions-mon-get-tablespace-get-table-space-metrics)
